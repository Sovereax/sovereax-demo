# Sovereax

AI outbound communication control layer.

We analyze AI-generated outbound messages before they are sent to customers — preventing spam, tone issues, and compliance risks.

## What it does
- Detects spam risk in outbound messages  
- Evaluates brand tone consistency  
- Checks compliance issues  
- Suggests improved message versions
